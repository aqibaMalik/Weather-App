## Installation

```
$ npm install
```

## App Configuration

Get Your OpenWeatherMap API KEY here: https://www.weatherapi.com/

⚠️ **Important Instructions** ⚠️:

- Create `.env` file in root of the project.
- Write the key you got from above url. _eg_  API_Key = <your_apikey>
